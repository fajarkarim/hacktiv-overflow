## URL path

#### main url:
```
http://localhost:8080
```

### login & register url:

URL | Description
------------ |
```/users/login```| user login
```/users/signup```| user signup


### user url:
URL | Description
------------ |
```/users/:username```| Get all user information



### question url:
URL | Description
------------ |
```/questions/:questionID```| get the question
```/questions/ask```| ask question


### search url:
URL | Description
------------ |
```/search/```| Get question information
